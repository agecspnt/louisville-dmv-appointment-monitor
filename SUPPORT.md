# Support

## Where to Ask for Help

- Use GitHub Issues for reproducible bugs and feature requests.
- Use GitHub Discussions (if enabled) for questions and usage help.

## What to Include

Please include the following in support requests:

- Operating system and version
- Node.js version (`node -v`)
- App version / commit
- Steps to reproduce
- Expected result vs actual result
- Relevant logs (remove secrets first)

## Response Expectations

This is a community-maintained project. Response times may vary.
